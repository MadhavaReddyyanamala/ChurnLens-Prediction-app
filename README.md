# ChurnLens-Prediction-app
An end-to-end Machine Learning web application that predicts whether a customer is likely to churn, built with a modern UI and deployed using Streamlit on Hugging Face.

# 🌐 Live Demo
👉 https://huggingface.co/spaces/mad0030/ChurnLens

# 🧠 Problem Statement
Customer churn is a major challenge for businesses. This project predicts customer churn based on behavioral and billing data, helping companies take proactive actions to retain customers.

#⚙️ Features
✨ Predict customer churn in real-time
📊 Interactive and user-friendly dashboard
📈 Displays churn probability
🧠 Uses trained Machine Learning model
⚡ Fast and responsive UI with Streamlit

# 🛠️ Tech Stack
Frontend / UI
Streamlit
Backend / ML
Python
Scikit-learn
Pandas
NumPy
Deployment
Hugging Face Spaces

# 🚀 How It Works
User enters customer details
Data is preprocessed using trained encoders & scaler
Model predicts churn
Result is displayed with probability
